# TraderToolsTH — Version 2: Gold Trading Desk

เว็บไซต์ชุดที่สองแยกจาก Version 1 โดยสมบูรณ์ ออกแบบตามภาพอ้างอิงด้วยโทน Navy, Gold และ Teal

## Typography

- ภาษาอังกฤษและตัวเลข: Noto Sans
- ภาษาไทย: Bai Jamjuree

## Home sections

- Hero พร้อมกราฟแท่งเทียนและปุ่มเริ่มใช้งาน
- Signal ล่าสุดพร้อม Entry, Take Profit, Stop Loss และ Confidence
- ข่าวสารตลาด
- ปฏิทินเศรษฐกิจ
- เครื่องมือคำนวณสำหรับเทรดเดอร์
- แถบจุดเด่นและความน่าเชื่อถือ

## Responsive

- Desktop: Hero สองส่วนและ Dashboard สามคอลัมน์
- Tablet: Dashboard สองคอลัมน์
- Mobile: เรียงหนึ่งคอลัมน์และตัดเฉพาะข้อมูลตารางที่ไม่จำเป็น

เปิด `home.html` ผ่าน Local server เพื่อดูหน้าเว็บไซต์
