# TraderToolsTH Website Versions

โฟลเดอร์นี้แยกเว็บไซต์ออกเป็น 2 ชุดอย่างอิสระ:

- `Version-1-Premium-Dashboard` — แบบเดิมโทนน้ำเงิน/ฟ้าและ Dashboard พื้นสว่าง
- `Version-2-Gold-Trading` — แบบใหม่โทนน้ำเงินเข้ม/ทองตามภาพอ้างอิง

เปิด `index.html` หรือเข้า `http://127.0.0.1:8099/` เพื่อเลือกดูแต่ละแบบ

ไฟล์คำสั่ง GLM และโฟลเดอร์เปล่าที่ไม่เกี่ยวกับการรันเว็บไซต์ถูกนำออกแล้ว
